### QueueClip v13.0 完整功能清单
#### 一、核心队列式剪贴板管理
1.  **先进先出队列存储**：最多支持50条剪贴内容，遵循队列逻辑管理，超出容量自动提示
2.  **双粘贴模式**：
    - 出队模式：粘贴队首内容后自动从队列移除
    - 保留模式：粘贴后内容保留在队列，自动追加递增序号
3.  **条目锁定机制**：单条/批量锁定条目，锁定内容不会被出队，粘贴时自动追加序号，优先级高于全局模式
4.  **手动入队**：一键将当前系统剪贴板内容加入队列
5.  **条目删除**：支持单条删除、批量选中删除，支持Delete快捷键快速删除
6.  **位置调整**：支持单条置顶/置底、批量置顶/置底
7.  **拖拽排序**：鼠标拖拽自由调整队列顺序，边缘自动滚动适配长列表
8.  **内容编辑**：弹窗编辑条目完整内容，支持多行文本，Ctrl+回车快速保存
9.  **重复内容过滤**：新入队内容与队尾内容一致时自动跳过，避免重复条目

#### 二、v13.0 粘贴安全与撤回体系
1.  **延迟出队保护**：粘贴后默认350ms再执行出队，彻底避免“已出队未粘贴”的问题
2.  **粘贴撤回功能**：支持最近5次粘贴记录撤回，撤回内容自动回到队首
3.  **粘贴失败兜底**：写入剪贴板失败时自动保留条目，并弹出提示
4.  **全场景覆盖**：界面按钮粘贴、双击粘贴、全局快捷键粘贴均享有安全保护
5.  **粘贴状态栏**：底部实时显示最近粘贴内容预览，附带可点击的撤回入口
6.  **粘贴历史栈**：内存维护最近5条粘贴记录，状态同步显示可撤回数量

#### 三、全局快捷键系统
1.  **默认全局快捷键**：
    - `Alt+C`：将当前选中内容入队
    - `Alt+V`：粘贴队首内容
    - `Alt+R`：清空整个队列
    - `Alt+E`：唤出/隐藏主窗口
2.  **原生快捷键接管**：可选全局接管`Ctrl+C`/`Ctrl+V`，替换为队列入队/队列粘贴逻辑
3.  **防抖机制**：300ms按键防抖，避免重复触发导致异常
4.  **权限适配**：启动自动检测管理员权限，无权限时弹出提示（全局快捷键依赖管理员权限）

#### 四、内容筛选与管理
1.  **标签体系**：为条目设置自定义标签，支持单条/批量设置、批量清除
2.  **标签筛选**：下拉筛选全部、无标签、自定义标签分类的内容
3.  **实时搜索**：输入关键词实时过滤条目，同时匹配内容与标签，带180ms输入防抖
4.  **内容预览**：列表显示精简预览（去除换行、超长截断），鼠标悬浮400ms后显示完整内容悬浮提示
5.  **序号计数器**：保留模式、锁定条目自动追加递增序号，支持一键重置全局序号

#### 五、灵动动画与交互体验
1.  **窗口过渡动画**：显示/隐藏时淡入淡出+轻微缩放弹性效果
2.  **条目插入动画**：新条目加入时绿色背景渐变闪烁，模拟滑入效果
3.  **条目删除动画**：删除条目时红色红晕闪烁后消失
4.  **拖拽呼吸灯**：拖拽排序时指示条呈现明暗渐变的呼吸灯效果
5.  **浮动通知卡片**：粘贴、删除、撤回等操作触发顶部Toast通知，自动淡入淡出
6.  **按钮交互动效**：悬停、按下时背景色平滑过渡
7.  **滚轮快速选择**：鼠标滚轮可上下切换选中条目，无需手动点击
8.  **锁定列反馈**：点击锁定列切换状态时，界面即时刷新反馈

#### 六、右键快捷菜单
1.  粘贴此条（不出队）
2.  复制到系统剪贴板
3.  编辑条目内容
4.  设置/清除标签
5.  锁定/解锁条目
6.  置顶到队首 / 置底到队尾
7.  删除选中条目

#### 七、数据持久化与稳定性
1.  **本地自动存储**：队列数据以JSON格式保存在用户目录，退出、修改时自动保存
2.  **启动自动恢复**：程序启动时自动加载历史队列内容
3.  **剪贴板安全读写**：复制/读取剪贴板带多次重试机制，确保读写成功率
4.  **线程安全**：所有队列操作内置线程锁，多线程（快捷键、UI）并发无异常
5.  **异常兜底**：注册退出钩子，程序异常退出时尽量保存队列数据

#### 八、界面与细节优化
1.  窗口始终置顶，唤出后自动聚焦
2.  列表奇偶行异色，提升可读性
3.  支持Ctrl+鼠标多选条目
4.  窗口支持自由缩放，设置最小尺寸限制
5.  完整的状态提示，实时显示队列当前容量
6.  退出二次确认，防止误操作关闭程序
